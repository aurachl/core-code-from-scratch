>### Get Number From String `Exercise` 🔢
**Using String Methods** 🔗
```javascript 
function getNumberFromString(s) {
  return Number(s
  .match(/\d/g)
  .join(''));
} 
```
**Using Array Methods** 📅
```javascript 
function getNumberFromString(s) {
  return Number(s
  .split('')
  .filter(number => number <= '9' && number >= '0')
  .join(''));
}
```
